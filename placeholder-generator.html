🎮 Generador de Placeholders Pixel Art

📋 Instrucciones:

  • Selecciona el estilo de placeholder deseado
  • Presiona "Generar Placeholder"
  • Una vez satisfecho, presiona "Descargar PNG"
  • Coloca el archivo en la carpeta correspondiente (mundo1/, mundo2/, etc.)
  • Si quieres, puedes usar estos placeholders para testing rápido

💡 Tips:

  • Estos placeholders son solo para testing
  • Reemplázalos con imágenes reales pixel art cuando tengas listas
  • Puedes usar Aseprite, GIMP, o Piskel para crear imágenes reales
  • Mantén proporciones cuadradas (256x256, 512x512, etc.)
